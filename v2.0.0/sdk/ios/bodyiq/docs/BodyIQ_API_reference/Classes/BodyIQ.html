  • beginObservingWearableController(_:failure:success:)

    Initiates BodyIQ to write to the datastore any unread data stored on the Wearable Device and to continue recording any new data until endObservingWearableController is called. Will be invoked automatically if addActivityListenerForWearableController is called before this method. It is an error to call this function on an unconnected wearable controller. Should not be called while currently observing the wearable or already subscribed to wearable events. Thread safety: May be called from any thread. No guarantees are made regarding dispatch queue or thread for success or failure blocks. Move to desired dispatch queue if needed.

    Declaration

        success: () -> Void)
