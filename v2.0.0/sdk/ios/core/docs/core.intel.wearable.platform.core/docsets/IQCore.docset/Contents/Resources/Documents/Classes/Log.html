  • enableFileLogging(logLevel:maximumFileSize:rollingFrequency:maximumNumberOfLogFiles:)

    Enables logging to the file system.

    The log file will be rolled if maximum file size or rolling frequency are reached.

    Declaration

    Swift

    public class func enableFileLogging(logLevel level: LogLevel = .Verbose, maximumFileSize: UInt64 = 1024 * 1024 * 1, rollingFrequency: NSTimeInterval = 60 * 60 * 24, maximumNumberOfLogFiles: UInt = 2)

    Parameters

    logLevel

    The desired logging level for file logging. This value may be different than the current logging level. In this case the value provided here acts as filter. Default value is Verbose.
    maximumFileSize

    The maximum file size specified in bytes. Default value is 1024 * 1024 * 1 (1 MB).
    rollingFrequency

    The log rolling frequency specified in seconds. Default value is 60 * 60 * 24 (one day).
    maximumNumbreOfLogFiles

    The maximum number of log files that are saved to the file system. Default value is 2.
