    Upload a file to the user blob store

    • Parameters:
    • filePath: path of the local file to upload
    • assetType: name of blob type uploading
    • success: call to let the application know if the upload started
    • failure: error response before/during upload+cellular
    • progress: progress call during upload and will get invoked multiple times
    • completed: response call after upload completed.
