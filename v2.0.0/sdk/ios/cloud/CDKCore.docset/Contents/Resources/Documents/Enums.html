  • SessionType

    A cloud request that results in a long-term cloud response (e.g. file download) is classified as a BACKGROUND_SESSION. This also implies that it is OK for the device to reboot or the app to go to sleep; when the long-term response is received completely the app notified via a callback. See the BLOB STORE API for more information.
