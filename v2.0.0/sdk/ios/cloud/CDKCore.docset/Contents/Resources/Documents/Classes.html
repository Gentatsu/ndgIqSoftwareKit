The following classes are available globally.

  • AuthCredentials

    The credential object that stores OAuth tokens received from the cloud after the user logs in. These credentials will be included in all future cloud requests that need authentication. In general, applications do not need to worry about the credentials stored here. They are managed (i.e. refreshed, stored, received) as needed by the cloud SDK.
