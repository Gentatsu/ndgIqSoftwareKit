Detailed Description

This is the main communication channel between the SDK and its users. The user should register to the MessageHandler with a MessageListener and receive messages through it. On application startup - the user should first register and then init the message handler in order to avoid loosing accumulated messages. On application shutdown - be sure to unregister and dispose.
